# Prerequisites


* Java 11
* Docker
* Docker Compose

# Getting Started

### Simple executable commands

* ./mvnw package && java -jar target/drones-service-docker-image-0.1.0.jar
* sudo docker build -t drones-service-docker-image .
* sudo docker-compose up --build
